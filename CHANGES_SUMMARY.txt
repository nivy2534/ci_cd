✨ FITUR BARU YANG DITAMBAHKAN:

1️⃣ DELETE TASK
   - Endpoint: DELETE /api/tasks/<id>
   - UI: Button merah di setiap task
   - Confirmation: Dialog sebelum delete

2️⃣ EDIT TASK
   - Endpoint: PUT /api/tasks/<id>
   - UI: Modal dialog yang elegan
   - Fields: Title + Description

3️⃣ TOGGLE STATUS
   - Endpoint: PATCH /api/tasks/<id>/status
   - UI: Checkbox untuk mark as completed
   - Visual: Garis coret + background hijau

🔍 ENDPOINTS API:

GET    /api/tasks              → Ambil semua tasks
POST   /api/tasks              → Buat task baru
PUT    /api/tasks/<id>         → Update task (title/desc)
DELETE /api/tasks/<id>         → Hapus task
PATCH  /api/tasks/<id>/status  → Toggle status

═══════════════════════════════════════════════════════════════════════════

💾 DATABASE SCHEMA:

CREATE TABLE tasks (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT,
    status VARCHAR(50) DEFAULT 'pending',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
